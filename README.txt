# 💰 Insurance Cost Prediction (ML + Deployment)

## Overview
Predicts medical insurance costs using Linear Regression with gradient descent and regularization.

## Features
- Gradient Descent (from scratch)
- Learning Rate Optimization
- Ridge & Lasso Regularization
- Feature Importance
- Interactive Web App (Streamlit)

## Visualizations
- Cost vs Iterations
- Predictions vs Actual

## Tech Stack
- Python, scikit-learn
- Streamlit
- Matplotlib

## Run Locally

pip install -r requirements.txt  
python src/train.py  
streamlit run app/app.py
